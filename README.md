# 🎲 dice_py
 
A simple command-line dice roller written in Python. Roll multiple dice at once and see the results displayed as ASCII art.
 
## Preview
 
```
How many dice should be rolled?: 3
 
┌─────────┐┌─────────┐┌─────────┐
│  ●   ●  ││         ││  ●      │
│    ●    ││    ●    ││         │
│  ●   ●  ││         ││      ●  │
└─────────┘└─────────┘└─────────┘
 
The total of the rolled dice are: 12
```
 
## Features
 
- Roll any number of dice at once
- Side-by-side ASCII art display
- Running total of all rolled values
- Play again prompt
## Requirements
 
- Python 3.x
## Usage
 
```bash
python dice.py
```
 
Follow the prompts to choose how many dice to roll and whether to play again.
